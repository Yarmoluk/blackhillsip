# Interactive Learning Graph

This interactive visualization displays all 200 concepts and 295 dependencies in the learning graph. Click on any concept to see its dependencies and dependents.

**Controls:**

- **Search** - Type a concept name to find and highlight it
- **Filter** - Select a category to show only those concepts
- **Layout** - Switch between hierarchical and force-directed layouts
- **Click** a node to see its details
- **Scroll** to zoom, **drag** to pan
